# Productivity-Task-Tracking-Tool
This project is a spreadsheet-based system I developed to track team productivity, assign tasks, and monitor sales performance.
## Features
- Task assignment tracking
- Productivity monitoring
- Sales data organization
- Managerial insights for team performance

## Tools Used
- Microsoft Excel / Google Sheets

## Purpose
This tool was created to improve team efficiency and organization in a real work environment.
